Notes on maps

G5

  • 1x Disabled makes the layout more symmetrical.

Warehouse

  • No spawns exist for slot 6, which is why I have it set to Nothing.
  • Disabling slots on this map does not seem to alter the weapon layout per se, but disabling 2 or more slots can produce some weird results.
  • If you really want to, try disabling 5+6. This puts slot 1 weapons in the slot 5 spawns with no other changes.
